# Churn Behaviour Analysis

End-to-end SQL and Python project analyzing user churn behavior, feature adoption, and customer retention patterns.

## Objective
This project aims to analyze user engagement patterns and understand the factors contributing to customer churn in a simulated SaaS platform.

## Tech Stack
- SQL Server
- SQL
- Python
- Pandas
- Matplotlib
- Statistics & Hypothesis Testing
